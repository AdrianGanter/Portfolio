# CyberChef: The Basics

## Overview

**CyberChef** is a browser-based data transformation and analysis tool commonly used in cyber security for decoding, encoding, converting, extracting, and analysing data.

It operates using **recipes**, where multiple operations can be chained together to process data efficiently.

---

# Key Concepts

## Operations

CyberChef contains hundreds of operations that can be combined to manipulate data.

Common examples include:

- Base64 encoding and decoding
- URL encoding and decoding
- Hex and decimal conversions
- ROT13 transformations
- Date and timestamp conversions
- Data extraction (IPs, URLs, email addresses)

## Recipes

A recipe is a sequence of operations applied to input data.

Recipes allow complex transformations to be performed quickly and repeatedly using a consistent workflow.

## Data Analysis Workflow

When using CyberChef:

1. Define the objective.
2. Load or paste the data.
3. Select appropriate operations.
4. Review the output and adjust as required.

# Common Use Cases

### Data Extraction

Used to identify and extract:

- IP addresses
- Email addresses
- URLs

### Data Decoding

Used to convert encoded data into a readable format:

- Base64
- Base58
- Base85
- URL encoding

### Data Conversion

Used to transform data between formats:

- Text ↔ Hex
- Text ↔ Decimal
- Date ↔ Unix Timestamp
