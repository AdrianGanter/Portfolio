# Splunk: The Basics

## Overview

Completed an introductory lab focused on understanding what Splunk is, how it processes log data, and how analysts use it to investigate events.

---

## What is Splunk?

Splunk is a SIEM (Security Information and Event Management) platform used to collect, store, search, and analyse log data from systems across an environment.

It provides a central location where analysts can investigate events, identify suspicious activity, and gain visibility into what is happening across networks and endpoints.

---

## Core Components

### Forwarder
A lightweight agent that collects logs from devices and sends them to Splunk.

### Indexer
Receives log data, processes it, and stores it in a searchable format.

### Search Head
The analyst interface used to search, filter, and investigate logs.
