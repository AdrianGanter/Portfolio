## Phishing Email Investigation

### Objective
Investigated a suspected phishing email within a virtual environment to identify indicators of compromise and assess the attached file.

### Email Overview
- **Sender:** `info@mutawamarine.com`
- **Recipient:** `webmaster@redacted.org`
- Email claimed a funds transfer had been completed and included an attached invoice.

### Initial Indicators
- Generic greeting addressed to the recipient email.
- Poor formatting and inconsistent text styling.
- Suspicious attachment:
  - `SWT_#09674321____PDF__.CAB`
- Compressed archive disguised as a PDF invoice.

### Email Source Analysis
- Identified originating IP address:
  - `192.119.71.157`
- Performed a WHOIS lookup.
- IP address was registered to **HostPapa**.

  

### Domain Authentication Checks
Validated the sender domain using DMARC analysis tools.

- **SPF:** Hard Fail (`-all`)
- **DMARC:** Quarantine policy (`p=quarantine`)

These records indicated the sender failed SPF validation and that unauthenticated emails should be quarantined.

### Attachment Analysis
Downloaded the attachment inside a virtual environment and generated its SHA-256 hash.

**SHA-256**
```
2e91c533615a9bb8929ac4bb76707b2444597ce063d84a4b33525e25074fff3f
```

### Malware Verification
Submitted the file hash to VirusTotal.

<img width="1358" height="692" alt="image" src="https://github.com/user-attachments/assets/10ecf99c-874a-4c7b-92fd-2764edfa3e13" />

Results:
- Detected as malicious by **50/63** security vendors.
- Classified as **Trojan.MSIL/Loki**.
- File format identified as a **RAR archive**, despite using a `.CAB` extension.

### Key Takeaways
- Identified common phishing indicators through email content and formatting.
- Performed email header and source IP analysis.
- Validated SPF and DMARC authentication records.
- Safely analysed a suspicious attachment in an isolated environment.
- Used SHA-256 hashing and VirusTotal to verify malware.
- Recognised how attackers disguise malicious archives as invoice documents to increase the likelihood of user execution.
